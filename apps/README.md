# Apps del curso

Esta carpeta se reserva para aplicaciones auxiliares versionadas como submodulos Git.

## Sugerencia inicial

- api-github-actions-demo: aplicacion base para practicas de pipeline, contenedores y despliegue.

## Ejemplo de alta de submodulo

```bash
git submodule add https://github.com/PythonistaMX/api-github-actions-demo.git apps/api-github-actions-demo
git submodule update --init --recursive
```
